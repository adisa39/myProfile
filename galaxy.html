Galaxy Mobile Game App

This project was created for programing tutorial and python coding practice.

The game is played by tryin to keep a space sheep on tack by moving the ship to left and right The objective;
Implementation of different Kivy framework function that show real time user interface update and
responsiveness as it respond to touch and button event command.

The Application also provide template to practice Android App deployment using Kivy-for-Android extensions
with virtualbox and Ubuntu OS. This project can also be used to learn code
organisation and code debugging.
Technologies: KIVY, UBUNTU, VIRTUALBOX, PYTHON, ANDROID-STUDIO
